# Github_MCP

โปรเจคสำหรับเรียนรู้และทดลองใช้งาน **GitHub MCP (Model Context Protocol)**

## 📁 โครงสร้างโปรเจค

```
Github_MCP/
├── .vscode/
│   └── mcp.json      # MCP configuration
├── welcome.html      # หน้า Welcome Page
└── README.md         # ไฟล์นี้
```

## 🚀 เริ่มต้นใช้งาน

1. Clone repository นี้

```bash
git clone https://github.com/0ARTKUNG0/Github_MCP.git
```

2. เปิดใน VS Code

```bash
code Github_MCP
```

3. ตั้งค่า GitHub Personal Access Token เมื่อมี prompt ขึ้นมา

## 🔧 การตั้งค่า MCP

ไฟล์ `.vscode/mcp.json` มีการตั้งค่า MCP สำหรับเชื่อมต่อกับ GitHub Copilot API
